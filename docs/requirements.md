# MdV 要求書

## 1. 概要

ローカル環境で使用するシンプルなMarkdownファイルビューアー。
macOS上でMarkdownファイルを手軽に閲覧することを目的とする。

## 2. 背景・目的

- Markdownファイルを閲覧するためにエディタやブラウザを起動するのは手間がかかる
- Finderから直接Markdownファイルを開いて、整形された状態で読みたい
- 軽量でシンプルな専用ビューアーが必要

## 3. 対象ユーザー

- macOSを使用する開発者・ライター
- Markdownファイルを日常的に扱うユーザー

## 4. 機能要求

### 4.1 ファイルを開く

| ID | 要求 | 優先度 |
|----|------|--------|
| FR-01 | メニューバーからMarkdownファイルを開ける | 必須 |
| FR-02 | キーボードショートカット（Cmd+O）でファイルを開ける | 必須 |
| FR-03 | ドラッグ&ドロップでファイルを開ける | 必須 |
| FR-04 | Finderの「このアプリケーションで開く」からファイルを開ける | 必須 |

### 4.2 Markdown表示

| ID | 要求 | 優先度 |
|----|------|--------|
| FR-05 | Markdownを整形されたHTMLとして表示する | 必須 |
| FR-06 | コードブロックにシンタックスハイライトを適用する | 必須 |
| FR-07 | 見出し・リスト・リンク・画像・テーブル等の標準的なMarkdown要素を正しく表示する | 必須 |

### 4.3 UI/UX

| ID | 要求 | 優先度 |
|----|------|--------|
| FR-08 | ファイル未選択時にウェルカム画面を表示する | 必須 |
| FR-09 | OSのダークモード設定に連動して表示を切り替える | 必須 |
| FR-10 | タイトルバーに開いているファイル名を表示する | 必須 |

## 5. 非機能要求

| ID | 要求 | 優先度 |
|----|------|--------|
| NF-01 | macOS上で動作する | 必須 |
| NF-02 | アプリの起動が高速である（3秒以内） | 必須 |
| NF-03 | 一般的なサイズ（〜1MB）のMarkdownファイルを遅延なく表示できる | 必須 |

## 6. 対象外

- Markdownの編集機能
- 複数ファイルの同時表示（タブ等）
- ファイルの自動リロード（ウォッチ機能）
- Windows / Linux対応
- アプリの自動アップデート
